Repositório dedicado ao upload da atividade de Programação Orientada a Objeto utilizando linguagem Java.

# Instruções para utilização

- Iniciar o XAMPP, e iniciar o Apache e MySQL (MySQL deverá estar na porta 3306)
- Criar um banco de dados no MySQL Workbench com o nome "banco";
> Caso seja necessário alterar no arquivo persistence.xml (src/main/META-INF/persistence.xml)
